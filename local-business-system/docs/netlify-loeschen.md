# Netlify-Projekt löschen

## Was?
Das alte Deployment `glowing-kataifi-ed4931.netlify.app` ist noch aktiv.
Der Script-Tag wurde in Session 4 entfernt, aber das Projekt existiert noch auf Netlify.

## So löschst du es (1 Minute)

1. Geh zu https://app.netlify.com
2. Klick auf das Projekt "glowing-kataifi-ed4931" (oder suche es)
3. Gehe zu **Site configuration** (oder Settings)
4. Ganz nach unten scrollen zu **"Delete this site"**
5. Bestätigen

## Warum?
- Spart einen aktiven Eintrag in deinem Netlify-Account
- Verhindert Verwirrung (die alte URL ist nicht mehr in Benutzung)
- Keine Sicherheitsrisiken durch alten Code auf einer fremden Domain
